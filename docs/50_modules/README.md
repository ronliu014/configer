# 50_modules 模块级文档

状态：Active

本目录保存工具实现模块的文档入口和摘要。模块文档用于连接项目级设计、游戏配置说明和具体代码实现，避免开发者直接在大量配置分析文档中迷路。

## 当前模块

- [equip](equip/README.md)：装备配置 MVP，当前首个业务线。

## 计划模块

- `item/`：物品与装备配套道具能力。
- `language/`：多语言文案表能力。
- `role/`：角色基础配置能力。
- `drop/`：掉落库与奖励配置能力。

## 模块文档职责

每个模块至少应有 `README.md`，说明模块定位、范围、依赖配置、开发入口和验收标准。游戏配置规则放在 `docs/40_game_config/<module>/`；工具模块入口只链接这些规则并说明实现边界。
