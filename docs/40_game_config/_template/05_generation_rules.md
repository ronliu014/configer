# 生成规则

状态：Active

本文记录从原 Excel 公式、策划规则或配置流程迁移出的 configer 生成规则。target 输出默认写入生成后的静态值，不依赖 Excel 公式继续计算。

## 规则清单

| 规则 ID | 输出字段 | 输入字段 | 来源 | 是否阻止输出 |
|---|---|---|---|---|
| `<RULE_ID>` | `<fieldKey>` | `<input fields>` | 原公式 / 策划规则 | 是 |

## 规则详情

### `<RULE_ID>`：规则名称

- 输出字段：`<fieldKey>`
- 来源类型：`generated`
- 输入字段：`<input fields>`
- 原 Excel 公式：仅用于追溯；如果无公式，写“无”
- 规则步骤：
  1. 读取输入字段。
  2. 执行明确的计算或拼接。
  3. 输出静态值。
- 异常处理：
  - 输入缺失时阻止输出。
- 样例：
  - 输入：`<input>`
  - 输出：`<output>`
