# ID 与编码规则

状态：Active

本文说明本模块 ID 如何生成、反解和检查冲突。编码规则必须能被测试用例验证。

## 编码规则

| ID | 输出字段 | 输入字段 | 规则 | 冲突处理 |
|---|---|---|---|---|
| `<idName>` | `<fieldKey>` | `<inputA>`, `<inputB>` | 拼接或计算规则 | 冲突时阻止输出 |

## 规则步骤

1. 读取必需输入字段。
2. 按固定顺序拼接或计算。
3. 生成目标 ID。
4. 检查当前表和依赖表中是否冲突。
