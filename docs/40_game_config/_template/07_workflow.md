# 配置流程

状态：Active

本文说明策划在本模块中完成一个配置对象的步骤。流程必须区分手填、系统生成和只读关联。

## 新增流程

1. 确认依赖表已存在。
2. 填写 `manual` 字段。
3. 触发 configer 生成 `generated` 字段。
4. 检查关联主键是否存在。
5. 查看变更预览。
6. 输出到 `targetRoot`。

## 编辑流程

1. 从列表选择已有对象。
2. 修改允许编辑的 `manual` 字段。
3. 重新计算受影响的 `generated` 字段。
4. 检查 diff 和校验结果。

## 常见失败

| 失败 | 原因 | 处理 |
|---|---|---|
| 主键冲突 | 生成 ID 已存在 | 修改输入维度或序号 |
| 关联缺失 | 目标表没有对应主键 | 补齐来源配置或修正引用 |
