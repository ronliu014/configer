# 校验规则

状态：Active

本文列出本模块必须执行的检查规则。规则必须可测试，错误提示必须能帮助策划定位问题。

## 规则级别

| 级别 | 行为 |
|---|---|
| `Error` | 阻止输出 |
| `Warning` | 允许输出，但必须提示 |
| `Info` | 仅提示 |

## 规则清单

| 规则 ID | 级别 | 检查对象 | 条件 | 提示 | 是否阻止输出 |
|---|---|---|---|---|---|
| `<RULE_ID>` | `Error` | `<table.field>` | 条件说明 | 用户可行动提示 | 是 |

## 样例

每条 `Error` 规则至少提供一个失败样例。重要 `Warning` 规则应提供一个通过样例和一个提示样例。
