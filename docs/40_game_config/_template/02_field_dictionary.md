# 字段字典

状态：Active

本文定义本模块字段的稳定逻辑 key、源 Excel 字段和输出策略。代码和 schema 必须使用逻辑 key，不得用 `Remark*` 作为业务判断依据。

## 字段清单

| 逻辑 key | 表 | 源字段 srcName | 中文名 | 类型 | 来源 | 可编辑 | 必填 | 默认值 | 输出策略 | 说明 |
|---|---|---|---|---|---|---|---|---|---|---|
| `<fieldKey>` | `<table>` | `<srcName>` | `<label>` | `int` / `string` | `manual` | 是 | 是 | 无 | 输出静态值 | 字段说明 |

## 字段来源

| 来源 | 含义 | target 行为 |
|---|---|---|
| `manual` | 策划手填 | 可编辑，输出静态值 |
| `generated` | configer 根据规则生成 | 不可手填，输出静态值 |
| `formula` | 原 Excel 公式 | 仅用于规则追溯，应迁移为 `generated` |
| `ref` | 关联字段 | 默认只读，按关系规则校验 |
| `imported` | 外部表导入 | 默认只读，由来源表或规则决定 |
| `hidden` | 隐藏字段 | 默认不编辑 |
| `deprecated` | 废弃字段 | 不作为新规则依据 |
